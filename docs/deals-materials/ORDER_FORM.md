## PETUNJUK PENGGUNAAN ORDER FORM

### Untuk Developer:

1. **Kirim form ini ke klien setelah:**
   - Deal closed
   - Invoice sudah dikirim
   - DP sudah diterima (untuk Opsi 2 & 3) atau sudah lunas (Opsi 1)

2. **Cara mengirim:**
   - Convert ke PDF
   - Kirim via email + WhatsApp
   - Request klien untuk print, isi, scan, kirim balik
   - Atau bisa isi digital (Google Forms/Typeform jika prefer)

3. **Follow up:**
   - Day 3: Reminder jika belum isi
   - Day 5: Call/WA untuk tanya progress
   - Day 7: Tegas reminder (sesuai kontrak, data harus lengkap max 7 hari)

4. **Setelah form diterima:**
   - Check kelengkapan data
   - Request missing data via WhatsApp
   - Konfirmasi tanggal mulai development
   - Update project timeline

### Tips:

✅ **Jangan mulai development** sebelum form ini lengkap terisi  
✅ **Simpan form** sebagai arsip (fisik + digital)  
✅ **Reference form** saat ada pertanyaan dari klien tentang requirement  
✅ **Update status** di bagian "Catatan untuk Developer" secara berkala
