## 1. RUANG LINGKUP PEKERJAAN

### 1.1 Pengembangan Website

**PENYEDIA LAYANAN setuju untuk mengembangkan dan menyerahkan:**

**A. Website Halaman Arahan** dengan fitur berikut:
- Bagian Hero dengan badge kepercayaan dan ajakan bertindak
- Bagian Inventaris Produk (tampilan katalog dengan fitur terjual habis)
- Bagian Kategori (Macbook, Laptop, Gaming, Ultrabook)
- Bagian Layanan (Beli, Trade-in, Layanan)
- Bagian Testimonial (integrasi ulasan pelanggan)
- Bagian FAQ (minimal 8 pertanyaan umum)
- Bagian Metode Pembayaran
- Bagian Kontak dengan perutean WhatsApp pintar
- Bagian Lokasi dengan integrasi Google Maps
- Footer dengan informasi perusahaan
- Tombol WhatsApp mengambang (menu kontak 3 arah)

**B. Spesifikasi Teknis:**
- Stack Teknologi: React 18 + TypeScript + Vite
- Komponen UI: Tailwind CSS + shadcn-ui
- Hosting: Vercel atau setara (uptime 99.9%)
- Domain: .com atau .id (pilihan klien)
- Desain responsif mobile (dioptimalkan untuk perangkat mobile)
- Waktu loading cepat (target < 3 detik)
- Dioptimalkan untuk SEO (dapat diindeks Google)
- Sertifikat SSL (keamanan HTTPS)

**C. Deliverables:**
- Website yang sepenuhnya fungsional (aktif dan dapat diakses)
- Registrasi dan konfigurasi domain
- Penyiapan hosting dan deployment
- Integrasi Google Analytics
- Repository kode sumber (GitHub repo pribadi)
- Dokumentasi dasar (prosedur pembaruan)
- Sesi pelatihan 30 menit untuk klien/staf

### 1.2 Layanan Pemeliharaan (Bulanan)

**PENYEDIA LAYANAN setuju untuk menyediakan pemeliharaan berkelanjutan:**

**A. Manajemen Konten:**
- Pembaruan produk (tambah, edit, hapus) - hingga 20 produk/bulan
- Unggahan dan penggantian foto - tanpa batas
- Pembaruan harga dan spesifikasi - tanpa batas
- Manajemen status terjual habis - tanpa batas
- Pembaruan konten teks (minor) - hingga 5 perubahan/bulan

**B. Pemeliharaan Teknis:**
- Perbaikan bug (jika ada)
- Update keamanan dan patch
- Optimisasi performa
- Update kompatibilitas browser
- Manajemen dan pemantauan hosting
- Manajemen pembaruan domain
- Pembaruan sertifikat SSL
- Manajemen backup (backup otomatis mingguan)

**C. Layanan Dukungan:**
- Dukungan WhatsApp/telepon selama jam kerja (9 AM - 6 PM WIB, Senin-Sabtu)
- Dukungan email (respons dalam 24 jam)
- Dukungan darurat untuk masalah kritis (respons dalam 2-4 jam)
- Laporan performa bulanan (traffic, konversi, rekomendasi)

**D. Pengecualian (Biaya Tambahan):**
- Perubahan desain besar atau rebranding
- Penambahan fitur baru di luar ruang lingkup asli
- Integrasi dengan sistem pihak ketiga (platform e-commerce, CRM, dll)

- Dapat menyederhanakan bahasa jika kedua belah pihak nyaman
- Bagian kunci: Ruang Lingkup, Pembayaran, Pembatalan, kepemilikan IP
- Bahkan perjanjian informal lebih baik daripada tidak ada perjanjian!

### Tanda Peringatan untuk Diwaspadai:
- Klien menginginkan revisi tanpa batas (risiko scope creep)
- Klien menolak untuk membayar biaya di muka apa pun (risiko pembayaran)
- Klien ingin memiliki semua kode tanpa pembayaran penuh (risiko sengketa IP)
- Ketentuan pembatalan yang tidak jelas (risiko penguncian)
