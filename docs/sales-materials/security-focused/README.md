# Security-Focused Sales Materials

## Overview

This folder contains sales materials that emphasize **security, risk mitigation, and protection** as the primary value proposition.

## Best Use Cases

Use these materials when:
- [+] Client has experienced security incidents (hack, fraud, account compromise)
- [+] Client is very concerned about security risks
- [+] Pain point is the strongest motivator
- [+] Client wants to prevent future security issues
- [+] Situation involves recent or ongoing security concerns

## Target Client Profile

- Businesses that have experienced Instagram/social media hacks
- Companies with high-value partnerships at risk (banks, marketplaces)
- Gold Merchant/Premium sellers on e-commerce platforms
- Businesses handling sensitive customer data
- Companies concerned about fraud and impersonation

## Tone & Messaging

**Primary Emotion:** Concern → Relief → Confidence
**Key Phrases:**
- "20 bulan lalu saat terjadi hack..."
- "Melindungi dari risiko..."
- "Benteng terakhir untuk bisnis Anda..."
- "Jangan sampai terulang lagi..."

## When NOT to Use

Avoid these materials when:
- [X] Client has NOT experienced security issues
- [X] Client is more motivated by growth/opportunity than fear
- [X] You want a generic template for multiple clients
- [X] Security is not a top concern for the client

→ In these cases, use **growth-focused** materials instead.

## Customization Tips

When using with specific clients:
1. Research their specific security incident details
2. Quantify their actual losses from past incidents
3. Reference their specific partnerships/status at risk
4. Adjust ROI calculations to their business size
5. Update timeline and pricing as needed

## Success Metrics

This approach works best when:
- Client acknowledges the past pain (security incident)
- They express concern about it happening again
- They have quantifiable losses from security issues
- They value protection over growth opportunities
